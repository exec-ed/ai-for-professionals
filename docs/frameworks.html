The 4 Frameworks

Practical tools for working with AI. Each framework answers a different question and is practised in a dedicated exercise.

1

AI Task Suitability Spectrum

Question: WHEN should I use AI for this task?

AI-Led ◄──────────────────────────────► Human-Only Routine Judgment Complex Empathy High-volume AI-supported Human-led Relationships Pattern-based Editing/review Sub-task help Novel judgment

Four zones:

  • AI-Led — Drafting, summarising, formatting, data entry
  • AI-Assisted — Analysis, research, editing, brainstorming
  • Human-Led, AI-Supported — Strategy, design, negotiation prep
  • Human-Only — Performance reviews, crisis leadership, creative vision
Practised in: Exercise 1 — AI Tool Test Drive (9:50 AM)
2

RTCF Prompt Framework

Question: HOW do I communicate effectively with AI?

R — Role Set the AI's expertise and perspective T — Task Define the specific action clearly C — Context Provide relevant background and constraints F — Format Specify the desired output structure

Key principle: The quality of your output is directly proportional to the quality of your input.

Practised in: Exercise 2 — Prompt Engineering Workshop (11:30 AM)
3

AI-Assisted Analysis Workflow

Question: How do I use AI for DATA work?

Prepare → Upload → Question → Verify → Refine → Present ▲ MOST IMPORTANT

Six steps:

  1. Prepare — Define the question before touching the tool
  2. Upload — Get data into the AI appropriately
  3. Question — Ask progressively deeper questions
  4. Verify — Cross-check against what you know
  5. Refine — Iterate based on findings
  6. Present — Format for your audience

Key principle: AI finds patterns fast, but humans decide what patterns mean.

Practised in: Exercise 3 — Data Analysis Challenge (1:45 PM)
4

AI Opportunity Finder

Question: WHERE does AI fit in my daily work?

Automate
High-volume, repetitive tasks: emails, scheduling, data entry
Assist
Information synthesis: research, summaries, meeting notes
Generate
Content creation: drafts, reports, presentations
Analyse
Data review: trends, patterns, anomaly detection

Key question: Which parts of my weekly work are high-volume, synthesis-heavy, content-heavy, or data-heavy?

Practised in: Exercise 4 — Your AI-Enhanced Workflow (3:20 PM)
